`cin`：\
1.忽略空格和换行符 \
2.当读取的类型与与">>"后的类型不一致时， `cin`对象中的一个**错误标记**被设置(**转化为bool值为false**)来中断输入， **欲在此输入要用`cin.clear();`重置错误标记**，使程序能够继续读取输入\
`cin.get(char)`：\
1.会读取每一个字符(**即使是空格，换行符等**)。\
2.它**返回**一个**cin对象**，**其可以转化为bool类型**，判断是否成功读到字符。\
`cin.get()`：\
1.**返回`int`类型的字符编码** \
2.当到EOF(End Of File)时，他会**返回符号常量**EOP(为-1)表示输出已尽。(**EOP之只能用int变量与之比较**)
```C++
int n; // input: 3
cin >> n; // output -> 3
n = cin.get(); // output -> 51 ('3'的ASCII码)
```
