**数组**在大多数表达式中会自动转换为指向其**首元素的指针**，这叫做"**数组到指针的衰减**"
```C++
#include <iostream>
using namespace std;
int main(){
  int a[5] = {1, 2, 3, 4};
  cout << *a; // input:1
  return 0;
}
```
在本例中，`a`就是指向第一个元素的指针

---
**指向数组的指针** vs **指针数组**
```C++
int (*p)[n]; // 生成指向 n个元素的数组 的指针(后面跟上数组的地址&array)
int *p[n]; // 生成容纳 n个指针 的数组，p指针指向这个数组
```
